Multi Cycle Processor Assembler and Simulator
Programmed a simulator for the processes of a multi-stage processor pipeline. The program works in two stages. The first stage takes an assembly file in LegV8 assembly language and assembles it into machine code in C language. Then the second stage uses the output from machine code and simulates a computer processor that takes in commands to update and access a virtual processor registry and stack. Debugged using many edge test cases that ensured that the program accounts for data and control hazards and stalls the pipeline for the fastest execution time possible.
